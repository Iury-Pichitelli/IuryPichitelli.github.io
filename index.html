Personal Statement

Meu primeiro contato com a tecnologia foi em 2021, quando comecei a aprender a linguagem de programação Python. Desde então, venho expandindo meus conhecimentos e, atualmente, estou focado no desenvolvimento de sites utilizando HTML5, CSS3 e JavaScript, além de aplicar controle de versionamento com Git e GitHub em meus projetos.

Atualmente, estou desenvolvendo um site voltado para o setor onde trabalho, com o objetivo de documentar os processos administrativos da área. Essa iniciativa busca reduzir a dependência do conhecimento individual dos colaboradores, facilitando a cobertura de férias, otimizando o suporte ao faturamento administrativo e promovendo uma maior organização dos processos internos.

Tenho como projeto futuro expandir essa solução para os demais setores da empresa, alcançando um número maior de usuários. Essa expansão visa mitigar ainda mais as dependências operacionais, padronizar os processos e contribuir significativamente para o desenvolvimento organizacional e a eficiência dos setores.
